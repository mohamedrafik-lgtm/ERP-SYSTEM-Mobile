═══════════════════════════════════════════════════════════════

🎉 ما الجديد؟

تم إنشاء قسم جديد منفصل في القائمة اسمه "إدارة الاختبارات"
يحتوي على جميع الصفحات المتعلقة بالاختبارات والأسئلة

═══════════════════════════════════════════════════════════════

📋 محتويات القسم الجديد:

   1️⃣ بنك الأسئلة (Questions)
   2️⃣ إضافة سؤال (AddQuestion)
   3️⃣ إدارة الاختبارات المصغرة (QuizManagement)
   4️⃣ إضافة اختبار مصغر (AddQuiz)

═══════════════════════════════════════════════════════════════

🚀 كيف تشوف التحديث؟

───────────────────────────────────────────────────────────────
الطريقة السريعة (الأفضل):
───────────────────────────────────────────────────────────────

1. افتح Terminal في مجلد المشروع

2. نفذ الأمر:
   
   npx react-native start --reset-cache

3. في terminal آخر:

   npx react-native run-android

4. افتح التطبيق → افتح القائمة (☰)

5. ستجد قسم جديد: "إدارة الاختبارات" 🎉

❓ القسم الجديد لا يظهر؟

───────────────────────────────────────────────────────────────
الحل 1: إعادة التشغيل مع مسح الكاش (الأضمن)
───────────────────────────────────────────────────────────────

npx react-native start --reset-cache

───────────────────────────────────────────────────────────────
الحل 2: تسجيل خروج ودخول
───────────────────────────────────────────────────────────────

1. افتح التطبيق
2. القائمة (☰) → تسجيل الخروج
3. أغلق التطبيق تماماً
4. افتح التطبيق وسجل دخول

───────────────────────────────────────────────────────────────
الحل 3: مسح بيانات التطبيق
───────────────────────────────────────────────────────────────

Settings → Apps → [تطبيقك] → Storage → Clear Data

───────────────────────────────────────────────────────────────
الحل 4: تحقق من دورك
───────────────────────────────────────────────────────────────

إذا كان دورك "محاسب" أو "موظف" → لن تظهر الصفحات
يجب أن تكون "مدير" أو أعلى
