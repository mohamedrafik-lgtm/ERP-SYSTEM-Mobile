# صلاحيات إدارة الكارنيهات

## الصلاحية المطلوبة:
- **Resource**: `dashboard.id-cards`
- **Action**: `manage`

## الوظائف التي تتطلب هذه الصلاحية:
1. عرض صفحة إدارة الكارنيهات
2. عرض إحصائيات الكارنيهات
3. البحث والفلترة في قائمة المتدربين حسب حالة الكارنيه
4. تحديث حالة تسليم الكارنيه
5. عرض تفاصيل طباعة وتسليم الكارنيهات

## ملاحظة:
- هذه الصلاحية منفصلة عن صلاحيات عرض المتدربين العادية
- تُعطى عادة لمسؤولي شؤون المتدربين أو من يدير الأوراق الرسمية
- يمكن دمجها مع صلاحيات أخرى حسب الحاجة
