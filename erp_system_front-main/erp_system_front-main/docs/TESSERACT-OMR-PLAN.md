## 🔧 الخوارزمية:

### **خطوة 1: معالجة الصورة**
```
1. تحويل لـ Grayscale
2. تطبيق Threshold (أبيض/أسود)
3. تحسين التباين
```

### **خطوة 2: قراءة الرموز بـ Tesseract**
```
1. Tesseract يقرأ كل النصوص في الصورة
2. يجد: A, B, C, D, ص, خ
3. يحدد مواقعها (X, Y)
```

### **خطوة 3: كشف الدوائر المظللة**
```
1. فحص المنطقة فوق كل رمز
2. حساب كثافة السواد
3. إذا > 40% = مظللة
```

### **خطوة 4: المطابقة**
```
لكل سؤال:
  - ابحث عن الرموز (A, B, C, D أو ص، خ)
  - تحقق أي دائرة فوقها مظللة
  - النتيجة = الرمز الذي دائرته مظللة
```
