# صفحة الرسوم والمستحقات للمتدربين 💰

## الوصف
صفحة جميلة ومتجاوبة لعرض رسوم المتدربين وحالة المدفوعات مع واجهة مستخدم حديثة ومتطورة.

## الميزات ✨

### 1. نظرة عامة مالية شاملة 📊
- **إجمالي الرسوم**: عرض المبلغ الكلي المطلوب
- **المبلغ المدفوع**: ما تم دفعه فعلياً
- **المبلغ المتبقي**: ما يجب دفعه
- **شريط التقدم**: نسبة مئوية للدفع مع تأثيرات بصرية متحركة

### 2. بطاقات رسوم محسنة 🎨
- **تصميم طبقي جديد** مع header منفصل
- **أيقونات ملونة** لكل نوع رسوم
- **حالة الدفع** مع ألوان مميزة:
  - 🟢 أخضر: مدفوع بالكامل
  - 🔵 أزرق: مدفوع جزئياً  
  - 🟡 أصفر: في انتظار الدفع
  - 🔴 أحمر: ملغي

### 3. إحصائيات سريعة لكل رسوم 📈
- **مقسم ثلاثي**: مدفوع | متبقي | نسبة الدفع
- **ألوان تمييزية** لكل قسم
- **حسابات تلقائية** دقيقة

### 4. ترتيب ذكي للمدفوعات 🔄
- **مرتبة حسب الأحدث** تلقائياً
- **عداد للمدفوعات** في العنوان
- **تصميم متجاوب** محسن

### 5. قسم طرق الدفع القادمة 💳
- **عرض تشويقي** لطرق الدفع الإلكترونية
- **فوري باي وMobile Wallets** مع أيقونات جذابة
- **تاريخ إطلاق متوقع** واضح

### 6. مساعدة ونصائح محسنة 🆘
- **قسمان منفصلان**: مساعدة | نصائح مالية
- **تصميم grid متجاوب**
- **نصائح عملية** للمتدربين

## التصميم 🎨

### نظام الألوان
- **الأساسي**: تدرجات الأخضر الزمردي والتيل
- **الثانوي**: ألوان الحالة (أخضر، أزرق، أصفر، أحمر)
- **الخلفية**: تدرجات ناعمة مع تأثيرات شفافية

### المكونات البصرية
- **بطاقات مظللة** مع تأثيرات hover
- **أيقونات معبرة** من Heroicons
- **أشرطة تقدم متحركة**
- **تدرجات لونية** جذابة
- **تصميم متجاوب** لجميع الأجهزة

## الحالات المختلفة 📊

### 1. يوجد رسوم
- عرض البطاقات التفاعلية
- إحصائيات مالية شاملة
- أزرار الإجراءات المناسبة

### 2. لا توجد رسوم
- رسالة ودودة للمستخدم
- شرح عن كيفية ظهور الرسوم
- معلومات مفيدة عن النظام

### 3. خطأ في التحميل
- رسالة خطأ واضحة
- زر إعادة المحاولة
- تصميم متسق مع باقي النظام

## التكامل مع النظام 🔗

### البيانات
- يستخدم `useTraineeProfile` hook للحصول على البيانات
- يقرأ من `traineePayments` array في بيانات المتدرب
- يحسب الإحصائيات تلقائياً

### التنقل
- مرتبط مع الصفحة الرئيسية للمتدرب
- يظهر في القائمة الجانبية
- متكامل مع navigation system

## الاستخدام 🚀

### للمطورين
```typescript
// البيانات المطلوبة في traineePayments
interface TraineePayment {
  id: number;
  amount: number;
  paidAmount: number;
  status: 'PAID' | 'PARTIALLY_PAID' | 'PENDING' | 'CANCELLED';
  createdAt: string;
  paidAt?: string;
  fee: {
    name: string;
    type: 'TUITION' | 'SERVICES' | 'TRAINING' | 'ADDITIONAL';
  };
}
```

### للمستخدمين
1. انتقل إلى "الرسوم والمستحقات" من القائمة
2. اعرض حالتك المالية في النظرة العامة
3. راجع تفاصيل كل رسوم في البطاقات
4. استخدم أزرار الدفع (قريباً) للرسوم المعلقة
