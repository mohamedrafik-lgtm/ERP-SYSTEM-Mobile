## 🔑 العوامل المؤثرة على الدقة

### 1. جودة التظليل ⭐⭐⭐⭐⭐

```
✅ ممتاز (98% دقة):
├── قلم رصاص 2B أو أغمق
├── تظليل كامل متساوٍ ●
├── ضغط جيد على القلم
└── لا علامات خارج الدائرة

⚠️ جيد (93% دقة):
├── قلم رصاص HB
├── تظليل جيد لكن ليس متساوياً
└── بعض العلامات الخفيفة

❌ ضعيف (70-85% دقة):
├── قلم جاف
├── تظليل خفيف جداً
├── علامات بدلاً من تظليل كامل
└── تظليل متعدد (محو وإعادة)
```

### 2. جودة الصورة ⭐⭐⭐⭐

```
✅ ممتاز (98% دقة):
├── ماسح ضوئي 300+ DPI
├── تباين عالي (أسود على أبيض نقي)
├── لا ظلال
├── محاذاة مثالية
└── لا تشويه

⚠️ جيد (93% دقة):
├── كاميرا هاتف 12MP+
├── إضاءة جيدة ومتساوية
├── تركيز واضح
└── محاذاة جيدة

❌ ضعيف (70-85% دقة):
├── كاميرا قديمة < 8MP
├── إضاءة سيئة أو ظلال
├── صورة مشوشة
└── انحراف كبير
```

### 3. نظافة الورقة ⭐⭐⭐

```
✅ ممتاز:
├── ورقة نظيفة تماماً
├── لا طيات أو تجاعيد
├── لا كتابات إضافية
└── لا بقع أو علامات

⚠️ جيد:
├── طيات خفيفة
├── بعض العلامات على الهامش
└── نظيفة في منطقة الدوائر

❌ ضعيف:
├── كتابات على الدوائر
├── بقع أو علامات محو
├── طيات تعبر الدوائر
└── ورقة ممزقة
```

## 🎓 أمثلة على الأخطاء الشائعة

### خطأ 1: علامة خفيفة تبدو كأنها مظللة

```
الورقة:      ○ ◌ ○ ○     [5]
               D C B A
                 ↑ علامة خفيفة (ليست مظللة)

AI قد يكتشف: "C" (خطأ ❌)
الصحيح: لا إجابة (تخطي)
```

### خطأ 2: تظليل ضعيف

```
الورقة:      ○ ○ ◔ ○     [7]
               D C B A
                   ↑ تظليل خفيف

AI قد يتخطاه أو يختار خياراً آخر
الحل: تظليل أقوى
```

### خطأ 3: محو غير كامل

```
الورقة:      ○ ◑ ● ○     [10]
               D C B A
                 ↑ تم المحو لكن بقيت آثار
                   ↑ الإجابة الصحيحة

AI قد يختار: "C" (خطأ ❌)
الصحيح: "B"
الحل: محو كامل ونظيف
```

### 2. معالجة الصورة قبل الإرسال

```typescript
// يمكن إضافة:
async function enhanceImage(imageDataUrl: string): Promise<string> {
  const img = new Image();
  img.src = imageDataUrl;
  await imageLoad(img);
  
  const canvas = document.createElement('canvas');
  canvas.width = img.width;
  canvas.height = img.height;
  const ctx = canvas.getContext('2d');
  
  // رسم الصورة
  ctx.drawImage(img, 0, 0);
  
  // زيادة التباين
  const imageData = ctx.getImageData(0, 0, canvas.width, canvas.height);
  increaseContrast(imageData, 1.2); // +20% contrast
  ctx.putImageData(imageData, 0, 0);
  
  // إزالة الضوضاء (اختياري)
  // applyGaussianBlur(ctx, 1);
  
  return canvas.toDataURL('image/jpeg', 0.95);
}
```
